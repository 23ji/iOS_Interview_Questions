#  24. Swift의 클로저와 함수의 차이점은 무엇인가요?

## 1️⃣ 클로저(Closure)와 함수(Function) 개념

### 함수 (Function)

특정 작업을 수행하는 **이름이 있는 코드 블록**

특징

* 이름 존재
* 재사용 목적
* 일반적인 로직 구성에 사용

예

```swift
// File: MathUtility.swift
// Layer: Utility

func add(a: Int, b: Int) -> Int {
    return a + b
}
```

---

### 클로저 (Closure)

**이름이 없는 함수**

즉

> **코드를 값처럼 사용할 수 있는 함수**

예

```swift
// File: SortExample.swift
// Layer: Feature

let addClosure = { (a: Int, b: Int) -> Int in
    return a + b
}
```

함수와 같은 일을 하지만

* **변수에 저장 가능**
* **다른 함수의 파라미터로 전달 가능**

---

## 2️⃣ 함수 vs 클로저 차이

| 항목    | 함수         | 클로저        |
| ----- | ---------- | ---------- |
| 이름    | 있음         | 없음         |
| 선언 위치 | 전역 / 타입 내부 | 변수 안       |
| 전달    | 직접 전달 어려움  | 값처럼 전달 가능  |
| 사용 목적 | 일반 로직      | 콜백 / 고차 함수 |

핵심

**클로저는 "코드를 값처럼 다루기 위해 존재"**

---

## 3️⃣ 클로저가 필요한 이유

Swift에서는 이런 코드 많이 봄

예

```swift
array.sorted { $0 < $1 }
```

이 코드에서

```
{ $0 < $1 }
```

이게 **클로저**

정렬 기준을 **함수 대신 전달**하는 것

즉

```
정렬 알고리즘 + 비교 방식
```

을 분리할 수 있음

---

## 4️⃣ 클로저 내부 동작 원리 (중요)

클로저의 가장 중요한 특징

**외부 변수를 캡처(capture) 가능**

예

```swift
// File: ClosureCaptureExample.swift
// Layer: Example

func makeIncrementer() -> () -> Int {
    
    var count = 0
    
    return {
        count += 1
        return count
    }
}
```

사용

```swift
let increment = makeIncrementer()

increment() // 1
increment() // 2
increment() // 3
```

여기서

```
count
```

는 원래 함수가 끝나면 사라져야 함

하지만

**클로저가 count를 캡처해서 유지함**

이게 **Closure Capture**

---

## 5️⃣ 클로저가 일급 객체(First-Class Citizen)인 이유

일급 객체 조건

1️⃣ 변수에 저장 가능
2️⃣ 함수의 파라미터로 전달 가능
3️⃣ 함수의 반환값으로 사용 가능

Swift 클로저는 이 3개 전부 가능

예

### 1️⃣ 변수 저장

```swift
let multiply = { (a: Int, b: Int) in
    a * b
}
```

---

### 2️⃣ 함수 파라미터 전달

```swift
// File: Calculator.swift
// Layer: Utility

func calculate(_ a: Int, _ b: Int, operation: (Int, Int) -> Int) -> Int {
    return operation(a, b)
}
```

사용

```swift
calculate(3, 4) { $0 + $1 }
```

---

### 3️⃣ 함수 반환

```swift
func makeAdder(_ value: Int) -> (Int) -> Int {
    
    return { number in
        number + value
    }
}
```

---

## 6️⃣ 함수형 프로그래밍에서 클로저 활용

Swift는 **함수형 프로그래밍 지원 언어**

대표적으로 사용하는 것

### 1️⃣ map

배열 변환

```swift
// File: MapExample.swift
// Layer: Example

let numbers = [1,2,3,4]

let doubled = numbers.map { $0 * 2 }
```

결과

```
[2,4,6,8]
```

---

### 2️⃣ filter

조건 필터

```swift
let evenNumbers = numbers.filter { $0 % 2 == 0 }
```

결과

```
[2,4]
```

---

### 3️⃣ reduce

값 하나로 축소

```swift
let sum = numbers.reduce(0) { $0 + $1 }
```

결과

```
10
```

이 모든 고차 함수가 **클로저 기반**

---

## 7️⃣ 실생활 비유

클로저

→ **요리 레시피 카드**

함수

→ **정식 메뉴**

예

식당에서

```
정렬 방법 알려주세요
```

하면

요리사가

```
이 기준으로 해주세요
```

레시피 카드 전달

이 레시피 카드가 **클로저**

---

## 📌 핵심 요약

* **함수** → 이름 있는 코드 블록
* **클로저** → 이름 없는 함수 (코드를 값처럼 사용)
* 클로저는 **변수 저장 / 파라미터 전달 / 반환 가능 → 일급 객체**
* 외부 변수를 **capture** 할 수 있음
* Swift 함수형 프로그래밍(map, filter, reduce)에 핵심적으로 사용됨
